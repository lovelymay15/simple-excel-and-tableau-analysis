# Data Analytics Projects

Excel and Tableau analysis tasks completed for the Introduction to Data Analytics course in 2025.

## Projects Overview

### 1. Hotel Bookings Analysis (Excel)
Analysis of hotel booking data using Microsoft Excel, featuring custom visualizations and statistical analysis to identify booking patterns, cancellation trends, and occupancy insights.

**Tools:** Microsoft Excel

**Key Focus Areas:**
- Booking patterns and trends
- Cancellation analysis
- Occupancy metrics

### 2. HR Data Analysis (Tableau)
Interactive dashboard and visual analysis of human resources data exploring employee attrition patterns across demographics, departments, and job roles.

**Tools:** Tableau

**Key Focus Areas:**
- Attrition rates by gender, department, and age group
- Job satisfaction and turnover correlation
- Education field impact on employee retention

**Dashboard Reference:** Interactive dashboard design referenced from [YouTube tutorial](https://www.youtube.com/watch?v=oAIubTqg-Kw)
